# Advent of FPGA

This repository contains hardware-based solutions for Advent of Code 2025 (Days 1, 3, and 4, Part 2 only). Day 1 is implemented in SystemVerilog, while Days 3 and 4 are implemented in Hardcaml.

* **Day 1:** Located in the Day1/ folder.
* **Days 3 & 4:** Located in the hardcaml/ folder.

Each subdirectory contains additional READMEs with specific details and instructions.
